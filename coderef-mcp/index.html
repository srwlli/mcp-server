v1.0.0 8 Tools Production

MCP Tools (8)

  • mcp__coderef__query
    Query CodeRef elements by reference or pattern
  • mcp__coderef__analyze
    Deep analysis: impact, coverage, complexity
  • mcp__coderef__validate
    Validate CodeRef reference format and structure
  • mcp__coderef__batch_validate
    Batch validation (sequential or parallel)
  • mcp__coderef__generate_docs
    Generate documentation for CodeRef elements
  • mcp__coderef__audit
    Audit elements for validation, coverage, performance
  • mcp__coderef__nl_query
    Natural language queries - ask questions in plain English
  • mcp__coderef__scan_realtime
    Real-time source code scanning with AST parser
