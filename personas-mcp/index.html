What is personas-mcp?

personas-mcp is an MCP (Model Context Protocol) server that provides expert "personas" - specialized knowledge and behavior patterns that give AI assistants deep domain expertise. Think of it like switching between different expert consultants, each with comprehensive knowledge in specific areas.

Key Features

Comprehensive Expertise

1000-6000+ line system prompts designed for agentic AI-to-AI interaction, not just human guidance.

Independent Personas

Each persona is standalone with complete domain knowledge - no hierarchical dependencies.

Easy Activation

Simple slash commands: /use-persona mcp-expert or shortcuts like /docs-expert

MCP Protocol Compliant

Full JSON-RPC 2.0 over stdio, compatible with Claude Code and all MCP clients.

Tool Integration

Personas can use tools from other MCP servers while applying their domain expertise.

Production Ready

Tested and validated with 100% success rate on all personas.

Quick Start

1. Installation

# Clone or navigate to the repository cd C:\Users\willh\.mcp-servers\personas-mcp # Install dependencies pip install -e . # Add to Claude Code config (.claude/settings.json): { "mcpServers": { "personas-mcp": { "command": "python", "args": ["C:\\Users\\willh\\.mcp-servers\\personas-mcp\\server.py"] } } } # Restart Claude Code

2. Activate a Persona

/use-persona mcp-expert # MCP protocol expert /use-persona docs-expert # Documentation expert /use-persona coderef-expert # CodeRef-MCP builder /use-persona nfl-scraper-expert # NFL data scraping expert /use-persona taylor # General purpose agent # Or use shortcuts: /docs-expert /coderef-expert /nfl-scraper-expert /taylor

3. Ask Expert Questions

# With mcp-expert active: How should I design a new MCP tool that needs to access a database? # AI responds with MCP-specific expertise, protocol patterns, and best practices

How It Works

  1. User activates a persona via MCP tool (use_persona)
  2. Persona returns comprehensive system prompt with domain expertise
  3. AI adopts the persona's knowledge, communication style, and problem-solving approach
  4. AI uses any available MCP tools with that persona's expertise applied
  5. Persona influences tool usage patterns and recommendations

Key Principle: Personas are context providers, not tool wrappers. They don't modify other tools - they influence how the AI uses them with specialized knowledge.
