Workflows Quick Reference

Feature Lifecycle (Full)
End-to-end feature development with planning, implementation, and archival
/create-workorder → /execute-plan → implement → /update-deliverables → /update-docs → /archive-feature
Context Experts
Create persistent domain experts for files/directories with deep context analysis
/suggest-experts → /create-expert → /activate-expert → /get-expert → /update-expert
Standards & Auditing
Extract coding standards from codebase and audit for compliance
/establish-standards → /audit-codebase → fix violations → /check-consistency
Multi-Agent Coordination
Parallel agent work with conflict prevention and verification
/generate-agent-communication → /assign-agent-task → agents work → /verify-agent-completion → /aggregate-agent-deliverables
LLM Multi-Query
Query multiple LLMs and consolidate responses for consensus
/llm-prompt → query ChatGPT, Claude, Gemini → paste responses → /consolidate
Documentation Generation
Generate POWER framework docs for any project
/generate-docs → /generate-user-guide → /generate-quickref
Project Inventory
Comprehensive project analysis across all dimensions
/quick-inventory or /inventory-manifest + /dependency-inventory + /api-inventory + /test-inventory
